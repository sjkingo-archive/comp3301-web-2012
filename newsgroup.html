Newsgroup access

The course newsgroup can be found at uq.itee.comp3301 (we use the same newsgroup for the shadow course too). There are two methods of accessing the newsgroup:

  • Through my.UQ myNewsgroups (note: no threading is available)
  • In your own client through sslnews

If you have a newsreader on your machine (Thunderbird, mutt, pan, ...) you can use sslnews to access the EAIT newsgroups.

Configuration for sslnews access

Server: sslnews.eait.uq.edu.au
Port: 563
SSL/TLS: Enabled
Username and Password: ITEE login (sXXXXXXX)

Thunderbird note: you must select “Always request authentication when connecting to this server”

Using a non SSL-enabled newsreader

Some newsreaders do not support SSL access, so to connect to the EAIT newsgroups you must use stunnel. A sample config is provided to you below, but no further support with be given if you have to use this method.

/etc/stunnel/stunnel.conf:
    client = yes
    [sslnews]
    accept = localhost:119
    connect = sslnews.eait.uq.edu.au:563
